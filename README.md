## Tech Stack

### Frontend:
- React.js
- Tailwind CSS

### Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication

##  Features

-  User Registration and Login
-  Create and View Posts
-  View Own Profile
-  Delete Your Own Posts
-  Fully responsive for mobile and tablet

##  Demo / Admin Access

-  You can use any dummy email ID and name to register a new user.
- No admin dashboard required.
